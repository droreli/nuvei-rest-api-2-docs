# Simply Connect (Legacy)

Simply Connect provides pre-built checkout components for faster integration with reduced development effort.

<Warning>
  For new integrations, evaluate [REST API 2.0](/rest-2/introduction) for greater flexibility.
</Warning>

## When to Use

- Maintaining existing Simply Connect integrations
- Need embedded checkout components
- Prefer simplified integration over custom UI

## Typical Features

- Pre-built checkout UI components
- Event-based callbacks for payment status
- Simplified 3D Secure handling
- Embedded payment forms

## Official Documentation

For complete Simply Connect documentation, visit:

**https://docs.nuvei.com/api/main/indexMain_v1_0.html**

## Recommended Alternative

For new integrations, [REST API 2.0](/rest-2/introduction) offers:
- Full control over UX
- Modern API design
- Comprehensive payment method support
