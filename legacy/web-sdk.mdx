# Web SDK (Legacy)

Nuvei's Web SDK provides client-side JavaScript utilities for building custom checkout experiences.

<Warning>
  For new server-to-server integrations, use [REST API 2.0](/rest-2/introduction) instead.
</Warning>

## When to Use

- Maintaining existing Web SDK integrations
- Require client-side payment collection with SDK helpers
- Building custom checkout UI with JavaScript

## Typical Capabilities

- Tokenization and payment creation
- 3D Secure device fingerprinting
- APM discovery and presentation
- Client-side validation

## Official Documentation

For complete Web SDK documentation, visit:

**https://docs.nuvei.com/api/main/indexMain_v1_0.html**

## Recommended Alternative

For new integrations, [REST API 2.0](/rest-2/introduction) offers:
- Simpler authentication
- Better operational workflows
- Modern RESTful design
