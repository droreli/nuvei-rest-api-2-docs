## Migration from REST 1.0

If you're migrating from REST API 1.0, here are the key changes:

| Aspect | REST 1.0 | REST 2.0 |
|--------|----------|----------|
| Authentication | Checksum in body | API key header |
| Session | Required for every call | Optional |
| Refund endpoint | `/refundTransaction` | `/payments/{id}/refunds` |
| Void endpoint | `/voidTransaction` | `/payments/{id}/voids` |
| Settle endpoint | `/settleTransaction` | `/payments/{id}/settles` |
| Transaction linking | Multiple IDs | Single `paymentId` |
