# Przelewy24

<Info>
  **Payment Method ID:** `apmgw_Przelewy24`

Przelewy24 (P24) is Poland's most popular online payment platform, aggregating multiple Polish banks and payment methods into a single integration.

## How Przelewy24 Works

```mermaid
sequenceDiagram
    participant C as Customer
    participant M as Merchant
    participant N as Nuvei
    participant P as Przelewy24
    participant B as Polish Bank
    
    C->>M: Select Przelewy24
    M->>N: Payment request
    N-->>M: Redirect URL
    M->>C: Redirect to P24
    C->>P: Select bank/method
    P->>B: Initiate transfer
    C->>B: Authorize
    B->>P: Confirmed
    P->>N: Success
    N->>M: Webhook (APPROVED)
```
