# EPS

EPS (Electronic Payment Standard) is Austria's leading online payment system, supported by all major Austrian banks.

## How EPS Works

```mermaid
sequenceDiagram
    participant C as Customer
    participant M as Merchant
    participant N as Nuvei
    participant E as EPS
    participant B as Austrian Bank
    
    C->>M: Select EPS
    M->>N: Payment request
    N-->>M: Redirect URL
    M->>C: Redirect to EPS
    C->>E: Select bank
    E->>B: Redirect
    C->>B: Authenticate
    B->>E: Confirmed
    E->>N: Success
    N->>M: Webhook (APPROVED)
```
