<Info>
  Updates an existing order with new amount, currency, or items. Useful for dynamic pricing scenarios like adding shipping costs or applying discounts.
</Info>

## When to Use

- Adding shipping costs after address entry
- Applying coupon/discount codes
- Updating cart items
- Changing the transaction amount
- Modifying order details before payment

## Checksum Calculation

```
SHA256(merchantId + merchantSiteId + clientRequestId + amount + currency + timeStamp + merchantSecretKey)
```

## Common Use Cases

### Adding Shipping Costs

```javascript
// After user enters shipping address
await updateOrder({
  sessionToken: sessionToken,
  orderId: orderId,
  amount: (productTotal + shippingCost).toFixed(2),
  currency: 'USD',
  items: [
    ...cartItems,
    { name: 'Express Shipping', price: shippingCost.toFixed(2), quantity: 1 }
  ]
});
```

### Applying Discount

```javascript
// After user applies coupon
const discount = calculateDiscount(couponCode);
const newTotal = Math.max(0, originalTotal - discount);

await updateOrder({
  sessionToken: sessionToken,
  orderId: orderId,
  amount: newTotal.toFixed(2),
  currency: 'USD'
});
```
