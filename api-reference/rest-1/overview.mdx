# REST API 1.0 Overview

Nuvei's REST API 1.0 is a comprehensive server-to-server API that provides full control over the payment flow. It uses POST requests with JSON payloads and supports all payment operations including transactions, tokenization, subscriptions, and payouts.

<Info>
  REST API 1.0 is the original Nuvei server API and is fully supported. For new integrations, consider [REST API 2.0](/api-reference/rest-2/overview) for a more RESTful approach with simplified endpoints.
</Info>

## Base URLs

<CodeGroup>
```text Sandbox

```

```text Production

```
</CodeGroup>

## Authentication

All REST API 1.0 requests require authentication using:
- **merchantId**: Your unique merchant identifier
- **merchantSiteId**: Your site identifier
- **timeStamp**: Request timestamp in `YYYYMMDDHHmmss` format
- **checksum**: SHA256 hash for request validation

### Checksum Calculation

The checksum is a SHA256 hash of concatenated values. The exact values depend on the endpoint.

**For getSessionToken:**
```
SHA256(merchantId + merchantSiteId + clientRequestId + timeStamp + merchantSecretKey)
```

**For payment operations:**
```
SHA256(merchantId + merchantSiteId + clientRequestId + amount + currency + timeStamp + merchantSecretKey)
```

<Tip>
  The `merchantSecretKey` should never be exposed in client-side code. Always calculate the checksum on your server.
</Tip>

## Response Format

All responses include common status fields:

```json
{
  "internalRequestId": 12345678,
  "status": "SUCCESS",
  "errCode": 0,
  "reason": "",
  "version": "1.0"
}
```

| Field | Description |
|-------|-------------|
| `status` | Overall request status: `SUCCESS` or `ERROR` |
| `errCode` | Error code (0 for success) |
| `reason` | Error description if status is `ERROR` |
| `internalRequestId` | Nuvei's internal request identifier |

## Typical Payment Flow

<Steps>
  <Step title="Open Order">
    Call `/openOrder` to create an order and get a session token.
  </Step>
  <Step title="Initialize Payment (Optional)">
    For 3DS, call `/initPayment` to fingerprint the card and check if 3DS is required.
  </Step>
  <Step title="Execute Payment">
    Call `/payment` with the session token and payment details.
  </Step>
  <Step title="Handle 3DS (If Required)">
    If the response requires 3DS, call `/authorize3d` and redirect the user. After completion, call `/verify3d`.
  </Step>
  <Step title="Capture (If Auth)">
    For Auth transactions, call `/settleTransaction` to capture when ready to ship.
  </Step>
</Steps>

## Error Handling

### Transaction Status Values

| Status | Description |
|--------|-------------|
| `APPROVED` | Transaction successful |
| `DECLINED` | Transaction declined by issuer |
| `ERROR` | Processing error occurred |
| `PENDING` | Awaiting async response (APMs) |
| `REDIRECT` | Customer redirect required |

### Common Error Codes

| Code | Description |
|------|-------------|
| 0 | Success |
| -1 | Invalid checksum |
| -5 | Invalid parameter |
| -6 | Missing required field |
| -7 | Session expired |
| -1100 | Insufficient funds |
| -1101 | Card expired |
| -1102 | CVV mismatch |

## Code Examples

<CodeGroup>
```javascript Node.js
const crypto = require('crypto');
const axios = require('axios');

const merchantId = 'YOUR_MERCHANT_ID';
const merchantSiteId = 'YOUR_SITE_ID';
const secretKey = 'YOUR_SECRET_KEY';

// Generate checksum
function generateChecksum(params) {
  const concatenated = params.join('');
  return crypto.createHash('sha256').update(concatenated).digest('hex');
}

// Get session token
async function getSessionToken() {
  const timeStamp = new Date().toISOString().replace(/[-:T.Z]/g, '').slice(0, 14);
  const clientRequestId = `req_${Date.now()}`;
  
  const checksum = generateChecksum([
    merchantId, merchantSiteId, clientRequestId, timeStamp, secretKey
  ]);

  const response = await axios.post(
    'https://ppp-test.nuvei.com/ppp/api/v1/getSessionToken',
    {
      merchantId,
      merchantSiteId,
      clientRequestId,
      timeStamp,
      checksum
    }
  );

  return response.data.sessionToken;
}
```

```python Python
import hashlib
import requests
from datetime import datetime

MERCHANT_ID = 'YOUR_MERCHANT_ID'
SITE_ID = 'YOUR_SITE_ID'
SECRET_KEY = 'YOUR_SECRET_KEY'

def generate_checksum(*args):
    concatenated = ''.join(args)
    return hashlib.sha256(concatenated.encode()).hexdigest()

def get_session_token():
    timestamp = datetime.now().strftime('%Y%m%d%H%M%S')
    client_request_id = f'req_{int(datetime.now().timestamp())}'
    
    checksum = generate_checksum(
        MERCHANT_ID, SITE_ID, client_request_id, timestamp, SECRET_KEY
    )

    response = requests.post(
        'https://ppp-test.nuvei.com/ppp/api/v1/getSessionToken',
        json={
            'merchantId': MERCHANT_ID,
            'merchantSiteId': SITE_ID,
            'clientRequestId': client_request_id,
            'timeStamp': timestamp,
            'checksum': checksum
        }
    )

    return response.json()['sessionToken']
```

```php PHP
<?php
$merchantId = 'YOUR_MERCHANT_ID';
$merchantSiteId = 'YOUR_SITE_ID';
$secretKey = 'YOUR_SECRET_KEY';

function generateChecksum(...$params) {
    $concatenated = implode('', $params);
    return hash('sha256', $concatenated);
}

function getSessionToken() {
    global $merchantId, $merchantSiteId, $secretKey;
    
    $timeStamp = date('YmdHis');
    $clientRequestId = 'req_' . time();
    
    $checksum = generateChecksum(
        $merchantId, $merchantSiteId, $clientRequestId, $timeStamp, $secretKey
    );

    $data = [
        'merchantId' => $merchantId,
        'merchantSiteId' => $merchantSiteId,
        'clientRequestId' => $clientRequestId,
        'timeStamp' => $timeStamp,
        'checksum' => $checksum
    ];

    $ch = curl_init('https://ppp-test.nuvei.com/ppp/api/v1/getSessionToken');
    curl_setopt($ch, CURLOPT_POST, true);
    curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
    curl_setopt($ch, CURLOPT_HTTPHEADER, ['Content-Type: application/json']);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    
    $response = json_decode(curl_exec($ch), true);
    curl_close($ch);
    
    return $response['sessionToken'];
}
?>
```
