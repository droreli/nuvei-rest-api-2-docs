# Refund Payment

Refund all or part of a settled payment.

## Path Parameters

<ParamField path="paymentId" type="string" required>
  The payment ID to refund
</ParamField>

## Request Body

<ParamField body="processingEntityId" type="string" required>
  Your merchant processing entity ID
</ParamField>

<ParamField body="amount" type="number" required>
  Refund amount (must be â‰¤ remaining refundable amount)
</ParamField>

<ParamField body="currency" type="string" required>
  Currency code (must match original payment)
</ParamField>

<ParamField body="comment" type="string">
  Internal note for the refund
</ParamField>

## Example Request

<CodeGroup>
```bash cURL
curl -X POST https://api-sandbox.nuvei.com/payment-api/payments/375011/refunds \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "processingEntityId": "1234567890",
    "amount": 50,
    "currency": "USD",
    "comment": "Customer return"
  }'
```

```javascript JavaScript
const response = await fetch(
  'https://api-sandbox.nuvei.com/payment-api/payments/375011/refunds',
  {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': 'YOUR_API_KEY'
    },
    body: JSON.stringify({
      processingEntityId: '1234567890',
      amount: 50,
      currency: 'USD',
      comment: 'Customer return'
    })
  }
);
```
</CodeGroup>

## Response

<ResponseField name="paymentId" type="string">
  Original payment ID
</ResponseField>

<ResponseField name="transactionId" type="string">
  Refund transaction ID
</ResponseField>

<ResponseField name="amount" type="number">
  Refunded amount
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code
</ResponseField>

<ResponseField name="transactionType" type="string">
  Will be `Credit`
</ResponseField>

<ResponseField name="result" type="object">
  Refund result
</ResponseField>


## Notes

<Note>
  - Multiple partial refunds are allowed
  - Total refunds cannot exceed original amount
  - Currency must match original payment
  - Only settled payments can be refunded
</Note>
