# Class Objects Reference

This comprehensive reference documents all objects used in Nuvei API requests and responses. Objects are reusable across multiple API endpoints.

---

## paymentOption

The `paymentOption` object contains payment method details. Structure varies based on payment type.

### Card Payment

```json
{
  "paymentOption": {
    "card": {
      "cardNumber": "4111111111111111",
      "cardHolderName": "John Smith",
      "expirationMonth": "12",
      "expirationYear": "2025",
      "CVV": "217",
      "threeD": {},
      "storedCredentials": {}
    }
  }
}
```

**card Object Fields:**

| Field | Type | Max | Required | Description |
|-------|------|-----|----------|-------------|
| `cardNumber` | string | 20 | Yes | Full card number (13-19 digits) |
| `cardHolderName` | string | 70 | Yes | Name as printed on card |
| `expirationMonth` | string | 2 | Yes | 2-digit month (01-12) |
| `expirationYear` | string | 4 | Yes | 2 or 4-digit year |
| `CVV` | string | 4 | Yes* | Card verification value |
| `ccTempToken` | string | 45 | No | Temporary token from Web SDK |
| `threeD` | object | - | No | 3D Secure data |
| `storedCredentials` | object | - | No | Credential-on-file data |
| `cardHolderNameVerification` | object | - | No | Name verification |

### User Payment Option (Saved Card)

```json
{
  "paymentOption": {
    "userPaymentOptionId": "1234567890",
    "card": {
      "CVV": "217"
    }
  }
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `userPaymentOptionId` | string | Yes | Saved payment method ID |
| `card.CVV` | string | Conditional | Required if `alwaysCollectCvv` enabled |

### Alternative Payment Method (APM)

```json
{
  "paymentOption": {
    "alternativePaymentMethod": {
      "paymentMethod": "apmgw_PayPal",
      "email": "user@example.com"
    }
  }
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `paymentMethod` | string | Yes | APM identifier (e.g., `apmgw_PayPal`) |
| Additional fields | varies | Conditional | APM-specific fields |

### External Token (Network Token/DPAN)

```json
{
  "paymentOption": {
    "card": {
      "externalToken": {
        "externalTokenProvider": "ApplePay",
        "mobileToken": "{Apple Pay token data}"
      }
    }
  }
}
```

| Field | Type | Description |
|-------|------|-------------|
| `externalTokenProvider` | string | `ApplePay`, `GooglePay`, `Visa`, `Mastercard` |
| `mobileToken` | string | Token data from wallet provider |
| `cryptogram` | string | Payment cryptogram |
| `eciIndicator` | string | ECI from wallet |

## threeD

3D Secure authentication configuration and data.

### Request (Input)

```json
{
  "threeD": {
    "methodCompletionInd": "Y",
    "version": "2.2.0",
    "notificationUrl": "https://your-site.com/3ds-callback",
    "merchantUrl": "https://your-site.com",
    "platformType": "02",
    "v2AdditionalParams": {
      "challengeWindowSize": "05",
      "challengePreference": "01",
      "deliveryEmail": "user@example.com",
      "deliveryTimeFrame": "01",
      "giftCardAmount": "0",
      "giftCardCount": "0",
      "giftCardCurrency": "",
      "preOrderDate": "",
      "preOrderPurchaseInd": "01",
      "reorderItemsInd": "01",
      "shipIndicator": "01",
      "rebillExpiry": "",
      "rebillFrequency": "",
      "challengeWindowSize": "05"
    },
    "browserDetails": {
      "acceptHeader": "text/html,application/xhtml+xml",
      "ip": "192.168.1.1",
      "javaEnabled": "false",
      "javaScriptEnabled": "true",
      "language": "en-US",
      "colorDepth": "24",
      "screenHeight": "1080",
      "screenWidth": "1920",
      "timeZone": "-300",
      "userAgent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64)..."
    },
    "account": {
      "age": "05",
      "lastChangeDate": "20240101",
      "lastChangeInd": "01",
      "registrationDate": "20230601",
      "passwordChangeDate": "20240101",
      "resetInd": "01",
      "purchasesCount6M": "10",
      "addCardAttempts24H": "0",
      "transactionsCount24H": "1",
      "transactionsCount1Y": "25",
      "cardSavedDate": "20230701",
      "cardSavedInd": "02",
      "addressFirstUseDate": "20230601",
      "addressFirstUseInd": "02",
      "nameInd": "01",
      "suspiciousActivity": "01"
    }
  }
}
```

**Top-Level Fields:**

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `methodCompletionInd` | string | Conditional | `Y`, `N`, `U` - fingerprint completion |
| `version` | string | No | Preferred 3DS version: `2.1.0`, `2.2.0` |
| `notificationUrl` | string | Yes (3DS2) | Callback URL for challenge result |
| `merchantUrl` | string | No | Your website URL |
| `platformType` | string | Yes | `01` (app), `02` (browser) |

**browserDetails Fields (Required for 3DS2):**

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `acceptHeader` | string | Yes | Browser Accept header |
| `ip` | string | Yes | Customer IP address |
| `javaEnabled` | string | Yes | `true` or `false` |
| `javaScriptEnabled` | string | Yes | `true` or `false` |
| `language` | string | Yes | Browser language (e.g., `en-US`) |
| `colorDepth` | string | Yes | Screen color depth (bits) |
| `screenHeight` | string | Yes | Screen height in pixels |
| `screenWidth` | string | Yes | Screen width in pixels |
| `timeZone` | string | Yes | Timezone offset in minutes |
| `userAgent` | string | Yes | Full browser user agent |

**v2AdditionalParams Fields:**

| Field | Type | Description |
|-------|------|-------------|
| `challengeWindowSize` | string | `01`-`05` (see table below) |
| `challengePreference` | string | `01` (no preference), `02` (no challenge), `03` (challenge requested), `04` (mandate challenge) |
| `deliveryEmail` | string | Email for electronic delivery |
| `deliveryTimeFrame` | string | `01` (electronic), `02` (same day), `03` (overnight), `04` (2+ days) |
| `shipIndicator` | string | `01` (cardholder address), `02` (verified address), `03` (different address), `04` (pickup), `05` (digital), `06` (travel), `07` (other) |
| `preOrderPurchaseInd` | string | `01` (available), `02` (future) |
| `reorderItemsInd` | string | `01` (first time), `02` (reorder) |

**Challenge Window Sizes:**

| Code | Dimensions |
|------|------------|
| `01` | 250 x 400 pixels |
| `02` | 390 x 400 pixels |
| `03` | 500 x 600 pixels |
| `04` | 600 x 400 pixels |
| `05` | Full screen |

**account Fields (Risk Indicators):**

| Field | Type | Description |
|-------|------|-------------|
| `age` | string | Account age: `01` (no account), `02` (created during txn), `03` (<30 days), `04` (30-60 days), `05` (>60 days) |
| `lastChangeDate` | string | Last account change (YYYYMMDD) |
| `lastChangeInd` | string | `01` (during txn), `02` (<30 days), `03` (30-60 days), `04` (>60 days) |
| `registrationDate` | string | Account creation date (YYYYMMDD) |
| `passwordChangeDate` | string | Last password change (YYYYMMDD) |
| `purchasesCount6M` | string | Purchases in last 6 months |
| `transactionsCount24H` | string | Transactions in last 24 hours |
| `transactionsCount1Y` | string | Transactions in last year |
| `cardSavedDate` | string | Date card was saved (YYYYMMDD) |
| `suspiciousActivity` | string | `01` (no suspicious activity), `02` (suspicious activity) |

**Response Fields:**

| Field | Type | Description |
|-------|------|-------------|
| `methodUrl` | string | URL for 3DS method (fingerprinting) |

| `cReq` | string | Challenge request data |
| `result` | string | `Y` (authenticated), `N` (failed), `A` (attempted), `U` (unavailable), `C` (challenge required) |

| `isLiabilityOnIssuer` | string | `1` = liability shifted |
| `flow` | string | `frictionless` or `challenge` |

## billingAddress

Customer billing information.

```json
{
  "billingAddress": {
    "firstName": "John",
    "lastName": "Smith",
    "email": "john.smith@example.com",
    "phone": "+1-555-123-4567",
    "address": "123 Main Street",
    "address2": "Apt 4B",
    "city": "New York",
    "state": "NY",
    "zip": "10001",
    "country": "US",
    "cell": "+1-555-987-6543",
    "county": "New York County"
  }
}
```

| Field | Type | Max | Required | Description |
|-------|------|-----|----------|-------------|
| `firstName` | string | 30 | Recommended | Customer first name |
| `lastName` | string | 40 | Recommended | Customer last name |
| `email` | string | 100 | Recommended | Email address |
| `phone` | string | 18 | No | Phone number |
| `address` | string | 60 | Conditional | Street address (required for AVS) |
| `address2` | string | 60 | No | Address line 2 |
| `city` | string | 30 | Conditional | City |
| `state` | string | 2 | Conditional | State/province code (US/CA) |
| `zip` | string | 10 | Conditional | Postal/ZIP code (required for AVS) |
| `country` | string | 2 | Recommended | ISO 3166-1 alpha-2 country code |
| `cell` | string | 18 | No | Mobile phone |
| `county` | string | 255 | No | County |

<Note>
  For US/UK transactions, `address` and `zip` are required for AVS (Address Verification).
</Note>

### cvv2Reply Codes

| Code | Description |
|------|-------------|
| `M` | CVV match |
| `N` | CVV does not match |
| `P` | Not processed |
| `U` | Issuer not certified |
| `S` | Service not supported |

### avsCode Values

| Code | Description |
|------|-------------|
| `A` | Address matches, ZIP doesn't |
| `B` | Street address match (international) |
| `C` | Street and postal not verified (international) |
| `D` | Street and postal match (international) |
| `E` | AVS error |
| `G` | Issuer does not participate (international) |
| `I` | Address not verified (international) |
| `M` | Street and postal match (international) |
| `N` | Neither address nor ZIP match |
| `P` | Postal match (international) |
| `R` | System unavailable |
| `S` | Service not supported |
| `U` | Information unavailable |
| `W` | 9-digit ZIP matches, address doesn't |
| `X` | Exact match (address + 9-digit ZIP) |
| `Y` | Address + 5-digit ZIP match |
| `Z` | 5-digit ZIP matches, address doesn't |
