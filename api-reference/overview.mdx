# API Reference

Complete reference for all Nuvei REST API 2.0 endpoints.

## Base URLs

| Environment | Base URL |
|-------------|----------|
| Sandbox | `https://api-sandbox.nuvei.com/payment-api` |
| Production | `https://api.nuvei.com/payment-api` |

## Authentication

All requests require an API key in the header:

```
x-api-key: YOUR_API_KEY
```

## Endpoints

### Payments

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | [/payments](/api-reference/payments/create-payment) | Create a new payment |
| `POST` | [/payments/{id}/fingerprint](/api-reference/payments/fingerprint) | 3DS fingerprint |
| `POST` | [/payments/{id}/challenge](/api-reference/payments/challenge) | 3DS challenge |
| `POST` | [/payments/{id}/refunds](/api-reference/payments/refund) | Refund a payment |
| `POST` | [/payments/{id}/voids](/api-reference/payments/void) | Void a payment |
| `POST` | [/payments/{id}/settles](/api-reference/payments/settle) | Settle an auth |

### Accounts

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | [/buyers/{id}/payment-tokens](/api-reference/accounts/get-payment-tokens) | Get saved cards |

## Common Headers

```http
Content-Type: application/json
Accept: application/json
x-api-key: YOUR_API_KEY
```

## Response Codes

| Code | Description |
|------|-------------|
| `200` | Success |
| `400` | Bad Request - Invalid parameters |
| `401` | Unauthorized - Invalid API key |
| `404` | Not Found - Resource doesn't exist |
| `422` | Unprocessable - Business logic error |
| `500` | Server Error - Retry later |

## Common Response Fields

```json
{
  "paymentId": "string",
  "transactionId": "string",
  "amount": "number",
  "currency": "string",
  "transactionType": "string",
  "result": {
    "status": "approved | declined | pending | error"
  }
}
```

## Rate Limits

| Endpoint | Limit |
|----------|-------|
| All endpoints | 100 requests/second |
| Batch operations | 10 requests/second |

<Note>
  Rate limits may vary by account. Contact support for higher limits.
</Note>
