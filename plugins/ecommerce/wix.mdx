# Wix Integration

<Info>
  **Platform:** Wix Stores / Wix Payments  
  **Integration:** Native app  
  **Best For:** Small businesses, simple stores  
  **Support:** Cards, some APMs
</Info>

Accept payments on your Wix website through Nuvei, supporting credit cards and popular payment methods.

## Requirements

- Wix Premium plan with e-commerce
- Nuvei merchant account
- Business verification complete

## Setup

### 1. Connect Nuvei

1. Go to Wix Dashboard → **Settings**
2. Select **Accept Payments**
3. Click **Connect** next to Nuvei
4. Enter Nuvei credentials

### 2. Configure Settings

| Setting | Description |
|---------|-------------|
| Business Name | Shown on customer statements |
| Currency | Primary currency |
| Test Mode | Enable for testing |

### 3. Payment Methods

Enable methods in your Nuvei portal, they'll appear in Wix:

- Credit/Debit cards
- PayPal (if enabled)
- Apple Pay
- Google Pay

## Checkout Experience

Wix provides a seamless checkout:

1. Customer adds to cart
2. Proceeds to checkout
3. Selects Nuvei payment
4. Enters card details
5. Completes purchase

## Order Management

### View Transactions

1. Go to **Orders** in Wix Dashboard
2. Select order
3. View payment details

### Refunds

1. Go to **Orders**
2. Select order to refund
3. Click **Refund** → **Full** or **Partial**
4. Confirm refund

## Testing

1. Enable Test Mode in payment settings
2. Use test cards
3. Place test orders
4. Disable Test Mode for production

## Limitations

<Warning>
  Wix integration has some limitations compared to direct API:
  - Limited APM support
  - No custom checkout styling
  - Wix handles payment flow
</Warning>

## Advanced Features

For advanced features, consider:

- **Wix Velo** - Custom code for more control
- **Direct API** - Full Nuvei capabilities
