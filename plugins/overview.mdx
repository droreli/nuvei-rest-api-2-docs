## Quick Start

### What You'll Need

Before installing any plugin, have ready:

1. **Nuvei Credentials**
   - Merchant ID
   - Merchant Site ID
   - Secret Key
   
2. **Environment Choice**
   - Sandbox (for testing)
   - Production (for live payments)

3. **Platform Requirements**
   - Supported platform version
   - PHP version (for self-hosted)
   - SSL certificate

### General Installation Steps

<Steps>
  <Step title="Download the Plugin">
    Get the plugin from your platform's marketplace or Nuvei's download page.
  </Step>
  
  <Step title="Install on Your Platform">
    Follow platform-specific installation instructions.
  </Step>
  
  <Step title="Configure Credentials">
    Enter your Nuvei Merchant ID, Site ID, and Secret Key.
  </Step>
  
  <Step title="Set Environment">
    Start with Sandbox mode for testing.
  </Step>
  
  <Step title="Configure Payment Methods">
    Enable cards, APMs, and wallets as needed.
  </Step>
  
  <Step title="Test Transactions">
    Process test payments to verify setup.
  </Step>
  
  <Step title="Go Live">
    Switch to Production mode when ready.
  </Step>
</Steps>

## Platform-Specific Guides

### Magento 2

```bash
# Install via Composer
composer require nuvei/payments-magento-2
bin/magento module:enable Nuvei_Payments
bin/magento setup:upgrade
bin/magento cache:flush
```

Configuration: **Stores** → **Configuration** → **Sales** → **Payment Methods** → **Nuvei**

### WooCommerce

1. Download from WordPress Plugin Directory
2. Upload via **Plugins** → **Add New** → **Upload Plugin**
3. Activate the plugin
4. Configure at **WooCommerce** → **Settings** → **Payments** → **Nuvei**

### Shopify

1. Visit Shopify App Store
2. Search for "Nuvei"
3. Click **Add app**
4. Follow the installation wizard
5. Enter API credentials

## Testing Checklist

Before going live, verify:

- [ ] **Successful card payment** - Standard card with CVV
- [ ] **3D Secure flow** - Complete authentication
- [ ] **APM payment** - At least one APM (e.g., PayPal)
- [ ] **Refund** - Full and partial refund
- [ ] **Saved cards** - Return customer payment
- [ ] **Webhook reception** - Order status updates
- [ ] **Error handling** - Declined card displays proper message

## Troubleshooting

<AccordionGroup>
  <Accordion title="Payments not appearing" icon="eye-slash">
    1. Verify credentials are correct (check for spaces)
    2. Confirm environment matches (sandbox vs production)
    3. Check payment methods are enabled in Nuvei portal
    4. Clear platform cache
  </Accordion>
  
  <Accordion title="Webhook not received" icon="bell-slash">
    1. Confirm webhook URL is publicly accessible
    2. Check firewall/security rules
    3. Verify HTTPS certificate is valid
    4. Check webhook settings in Nuvei portal
  </Accordion>
  
  <Accordion title="3D Secure failing" icon="shield-halved">
    1. Ensure returnURL is configured correctly
    2. Check popup blockers aren't interfering
    3. Verify billing address is being sent
    4. Test with a different test card
  </Accordion>
</AccordionGroup>
