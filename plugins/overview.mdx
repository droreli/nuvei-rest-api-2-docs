## Plugin Features

All Nuvei plugins include:

| Feature | Description |
|---------|-------------|
| Card Payments | Visa, Mastercard, Amex, and more |
| 3D Secure 2.0 | PSD2 compliant authentication |
| APMs | Local payment methods by region |
| Tokenization | Save cards for returning customers |
| Refunds | Process refunds from admin panel |
| Webhooks | Automatic order status updates |
| Multi-currency | Accept payments in local currencies |
| Sandbox Mode | Test before going live |

## Installation Overview

<Steps>
  <Step title="Download Plugin">
    Get the plugin from your platform's marketplace or Nuvei's GitHub
  </Step>
  <Step title="Install">
    Upload via your platform's admin panel or composer/npm
  </Step>
  <Step title="Configure">
    Enter your Nuvei credentials (Merchant ID, Site ID, Secret Key)
  </Step>
  <Step title="Test">
    Enable sandbox mode and process test transactions
  </Step>
  <Step title="Go Live">
    Switch to production credentials
  </Step>
</Steps>

## Platform-Specific Guides

### Magento 2

```bash
# Install via Composer
composer require nuvei/payments-magento-2
bin/magento module:enable Nuvei_Payments
bin/magento setup:upgrade
bin/magento cache:flush
```

Configuration: **Stores** → **Configuration** → **Sales** → **Payment Methods** → **Nuvei**

### WooCommerce

1. Download from WordPress Plugin Directory
2. Upload via **Plugins** → **Add New** → **Upload Plugin**
3. Activate the plugin
4. Configure at **WooCommerce** → **Settings** → **Payments** → **Nuvei**

### Shopify

1. Visit Shopify App Store
2. Search for "Nuvei"
3. Click **Add app**
4. Follow the installation wizard
5. Enter API credentials
