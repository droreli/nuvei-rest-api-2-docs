## User Roles & Permissions

Access to Control Panel features depends on your assigned role:

| Role | Access Level |
|------|--------------|
| **Management** | Full access to all menu items, reports, and actions |
| **Finance_Actions** | Finance section + Transaction/Batch ID/Withdrawal reports |
| **Operations_Actions** | Operations menu, execute refunds/payouts/voids |
| **Risk_Actions** | Transaction data, risk reports, risk-related actions |
| **Risk Dispute** | Chargeback, Case Management, Disputes Reports only |
| **Developer only** | REST API 2.0 management, technical configuration |
| **Finance_ReadOnly** | View-only access to finance reports |
| **Operations_ReadOnly** | View-only access to operations reports |
| **Risk_ReadOnly** | View-only access to risk reports |

<Note>
Custom roles can be created by administrators from the **Settings > Team > Roles** tab.
</Note>

## Getting Your Credentials

Your API credentials are located in **Settings > My Integration Settings**:

<Steps>
  <Step title="Navigate to Settings">
    Click on **Settings** in the left navigation menu
  </Step>
  <Step title="Open Integration Settings">
    Select **My Integration Settings**
  </Step>
  <Step title="Find Your Credentials">
    - **Merchant ID**: Displayed at the top
    - **Merchant Site ID**: Listed for each site
    - **Secret Key**: Click "Show" (requires Admin rights)
  </Step>
</Steps>

<Warning>
Never share your Secret Key. It's used to authenticate API requests and calculate checksums.
</Warning>

The Control Panel header displays your current environment:

</Tip>

## Quick Actions

Common tasks you can perform:

| Task | Location | Description |
|------|----------|-------------|
| Search transactions | Operations > Transactions | Find by ID, date, amount, status |
| Issue refund | Transaction details > Actions | Refund after settlement |
| Void transaction | Transaction details > Actions | Cancel before settlement |
| Configure webhooks | Settings > Integration Settings | Set DMN URLs |
| View chargebacks | Risk > Chargebacks | Monitor and dispute chargebacks |
| Generate reports | Any report page | Export to CSV/XLSX |
