## Key Risk Metrics

### Chargeback Ratio

Monitor your chargeback ratio against card network thresholds:

| Network | Monitoring | Excessive | Program |
|---------|------------|-----------|---------|
| **Visa** | 0.65% | 0.90% | VDMP |
| **Mastercard** | 0.50% | 1.00% | ECP |

<Warning>
Exceeding thresholds can result in fines, higher processing fees, or account termination.
</Warning>

### Fraud Ratio

Track fraud as a percentage of total transactions:

| Metric | Calculation |
|--------|-------------|
| **Fraud Count Ratio** | Fraud transactions / Total transactions |
| **Fraud Amount Ratio** | Fraud amount / Total amount |

## Pre-Chargeback Prevention

### Ethoca Alerts

Real-time fraud alerts from issuers:

| Feature | Benefit |
|---------|---------|
| **Early Warning** | Know about fraud before chargeback |
| **Auto-Refund** | Automatically refund confirmed fraud |
| **Deflection** | Prevent chargeback filing |

Configure in **Risk > Risk Configurations > Ethoca Alerts**.

### Visa Rapid Dispute Resolution (VRDR)

| Feature | Description |
|---------|-------------|
| **Auto-Accept** | Automatically accept eligible disputes |
| **Threshold Rules** | Set auto-accept amount limits |
| **Liability Management** | Reduce representment costs |

## AVS/CVV Configuration

Configure Address Verification (AVS) and CVV validation:

### AVS Settings

| Code | Meaning | Recommended Action |
|------|---------|-------------------|
| A | Address match, ZIP no match | Review |
| N | No match | Decline |
| Y | Full match | Accept |
| Z | ZIP match, address no match | Review |
| U | Unavailable | Accept with caution |

### CVV Settings

| Code | Meaning | Recommended Action |
|------|---------|-------------------|
| M | Match | Accept |
| N | No match | Decline |
| P | Not processed | Review |
| U | Unknown | Review |

Configure in **Risk > Risk Configurations**.

## Best Practices

<AccordionGroup>
  <Accordion title="Monitor chargeback ratio weekly">
    Track your ratio trend to catch issues before exceeding thresholds:
    - Set up alerts at 0.5% (warning)
    - Investigate immediately above 0.7%
    - Implement action plan above 0.9%
  </Accordion>
  
  <Accordion title="Enable pre-chargeback tools">
    Use Ethoca and VRDR to prevent chargebacks before they're filed.
  </Accordion>
  
  <Accordion title="Respond to disputes promptly">
    - Visa: 30 days to respond
    - Mastercard: 45 days to respond
    - Missing deadlines = automatic loss
  </Accordion>
  
  <Accordion title="Maintain transaction records">
    Keep for dispute evidence:
    - Customer communications
    - Delivery confirmation
    - IP addresses and device info
    - 3DS authentication results
  </Accordion>
</AccordionGroup>
