# Case Management

Case Management provides a unified console for handling all dispute-related operations, including chargebacks, pre-chargebacks, and fraud investigations.

---

## Accessing Case Management

Navigate to **Risk > Case Management** in the Control Panel.

---

## Case Types

| Type | Description |
|------|-------------|
| **Pre-Chargeback** | Early warning fraud alerts |
| **Chargeback** | Formal disputes |
| **Pre-Arbitration** | Second-stage disputes |
| **Arbitration** | Network arbitration |
| **Fraud Investigation** | Internal fraud cases |

---

## Case Dashboard

### Overview Metrics

| Metric | Description |
|--------|-------------|
| Open Cases | Currently active cases |
| Due Today | Cases requiring action today |
| Overdue | Cases past deadline |
| Resolved This Month | Monthly resolution count |
| Win Rate | Percentage of won disputes |

### Case Queue

| Column | Description |
|--------|-------------|
| Case ID | Unique case identifier |
| Type | Case type |
| Transaction ID | Related transaction |
| Amount | Disputed amount |
| Due Date | Response deadline |
| Assigned To | Team member assigned |
| Status | Case status |
| Priority | High, Medium, Low |

## Case Workflow

<Steps>
  <Step title="Case Created">
    Case enters queue (auto or manual)
  </Step>
  <Step title="Assignment">
    Assign to team member
  </Step>
  <Step title="Investigation">
    Gather evidence and information
  </Step>
  <Step title="Decision">
    Accept, dispute, or escalate
  </Step>
  <Step title="Response">
    Submit response with evidence
  </Step>
  <Step title="Resolution">
    Track outcome
  </Step>
</Steps>

## Managing Cases

### Assign Case

<Steps>
  <Step title="Select Case">
    Click on case to open
  </Step>
  <Step title="Click Assign">
    Click Assign button
  </Step>
  <Step title="Select User">
    Choose team member
  </Step>
  <Step title="Add Notes">
    Optional assignment notes
  </Step>
</Steps>

### Add Evidence

<Steps>
  <Step title="Open Case">
    Select case from queue
  </Step>
  <Step title="Go to Evidence Tab">
    Click Evidence tab
  </Step>
  <Step title="Upload Files">
    Add supporting documents
  </Step>
  <Step title="Add Notes">
    Describe each piece of evidence
  </Step>
</Steps>

### Submit Response

<Steps>
  <Step title="Review Evidence">
    Verify all evidence is attached
  </Step>
  <Step title="Select Action">
    Accept, Dispute, or Partial
  </Step>
  <Step title="Add Response Notes">
    Explain your position
  </Step>
  <Step title="Submit">
    Submit response to network
  </Step>
</Steps>

## Case Status

| Status | Description |
|--------|-------------|
| **New** | Just created |
| **Assigned** | Assigned to team member |
| **In Progress** | Under investigation |
| **Pending Response** | Awaiting response submission |
| **Response Submitted** | Submitted to network |
| **Resolved - Won** | Ruled in merchant favor |
| **Resolved - Lost** | Ruled against merchant |
| **Escalated** | Moved to next stage |
| **Closed** | No further action |

---

## Case Priority

| Priority | Criteria | SLA |
|----------|----------|-----|
| **High** | Large amount, deadline soon | Same day |
| **Medium** | Standard case | 2 business days |
| **Low** | Small amount, time available | 5 business days |

## Automation Rules

Configure automatic case handling:

### Auto-Accept Rules

| Rule | Description |
|------|-------------|
| Amount threshold | Auto-accept under amount |
| Reason code | Auto-accept specific codes |
| Network | Auto-accept by network |

### Auto-Assignment Rules

| Rule | Description |
|------|-------------|
| Round robin | Distribute evenly |
| By expertise | Route by case type |
| By region | Route by geography |
